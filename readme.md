# Верстка главной страницы магазина Modimal на Gulp сборке

Технологии:

- HTML,
- SCSS,
- Gulp,
- JS

Установить зависимости:

```
npm i
```

Зпустить в режиме разработки:

```
gulp
```

Собрать версию для публикации:

```
gulp docs
```

[Смотреть демо](https://ekaterinaskirdina.github.io/Modimal_Gulp/)
